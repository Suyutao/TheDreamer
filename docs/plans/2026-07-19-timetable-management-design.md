# 课程表展示与统一管理设计

## 目标

修复课程表缺少管理入口、iPhone 周网格可以任意横向漂移、Today 空状态无法执行下一步的问题。课程表继续由 SwiftData 保存，系统日历不作为数据来源。

## 黑板贴业务映射

黑板贴中值得保留的行为包括：按开始时间排列当天课程、自动显示当天课程、明确编辑和删除单节课程、为课程设置开始前提醒。The Dreamer 使用现有 `Course`、`Timetable`、`ClassPeriod` 和 `CourseSchedule` 表达这些行为，不复制固定七天数组和 JSON 文件结构。

黑板贴的分隔线属于展示属性。The Dreamer 已经通过 `ClassPeriod` 保存课程起止时间，课程列表按时间自然形成间隔，因此不新增 `IsSplitBelow` 字段。黑板贴的强下课提醒依赖全屏 Windows 窗口，不适合 Apple 平台；The Dreamer 保留每条课程安排独立启用提醒和提前分钟数的模型。

## 展示结构

### iPhone 和窄窗口

课程表首页显示当前选中日期所在周的七天日期栏，下面是单日课程列表。点击日期直接切换，左右滑动页面切换前后日期。课程通过 `TimetableResolver.lessons(on:)` 解析并按 `ClassPeriod.startMinute` 排列。

当天没有课程时使用 `ContentUnavailableView`，提供进入统一管理页的原生操作。课程行显示课程名、起止时间、教师和地点，点击后进入课程详情。

### iPad 和 macOS 宽窗口

可用宽度达到 900 点时显示七天周视图。节次列使用固定宽度，其余七列平均分配可用空间。视图只允许纵向滚动，不使用横向与纵向组合的 `ScrollView`。宽度不足时自动采用按天页面。

## 统一管理

课程表首页工具栏和 Today 空状态都进入 `ScheduleManagementView`。管理页使用 `List`，包含课程表、课程和当前课程表安排三个 section。

- 课程表：创建、编辑名称与日期范围、设置首周单双周、设为当前、删除。
- 节次：在课程表详情中创建、编辑起止时间和顺序、删除。
- 课程：创建、编辑名称、SF Symbol 和可选关联科目、删除。
- 安排：选择课程、星期、节次、每周或单双周、教师、地点和提醒提前量、删除。

编辑器使用 `Form` 和 item-driven sheet。新增对象只在保存成功后插入；编辑对象在用户确认保存后更新。保存错误显示在当前编辑器中。删除课程表会级联删除节次和安排但保留课程；删除课程会保留安排并清空课程关联，执行前需要确认。
